Retrieve a Specific Case
^^^^^^^^^^^^^^^^^^^^^^^^

Send a request in the following format to retrieve data for a specific Case:

.. code::

    GET /v3/cases/{caseId}

For example, the following request will retrieve data for the Case whose ID is 3:

.. code::

    GET /v3/cases/3

JSON Response:
